Typography

Two fonts: DM Sans for all UI/utility text; Playfair Display italic only for heritage callouts ("Since 1947," pull quotes, sublines).

In business since 1947. We know New Orleans homes — older kitchens, tight staircases, shotgun layouts. We coordinate specialty installation when standard delivery won't work.
Section Eyebrow
DM Sans 700 / 11px / 0.14em UC
Shop by appliance
CTA Button
DM Sans 700 / 13px / 0.06em UC
Shop In-Stock Appliances

Rules for Playfair Display italic:

  • Use only for "Since 1947." / "In business since 1947" type statements
  • Use for pull quotes (full reviews from customers)
  • Use for the logo tagline in the footer
  • Never use for navigation, body copy, or UI controls
  • Limit to one Playfair instance per viewport at a time
